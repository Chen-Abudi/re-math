# Concept System

This system turns my learning journey into structured content.

## Goal

To capture insights, mistakes, and ideas from learning math and transform them into meaningful posts and articles.

---

## Process

1. Capture → ideas, mistakes, insights
2. Structure → organize into a clear message
3. Publish → LinkedIn, Medium (Maybe some other platforms)
4. Reflect → learn from feedback

---

## Content Pillars

- Thinking (logic, reasoning, clarity)
- Learning System (notes, exercises, mistakes)
- Real-World Application (money, cooking, decision, etc.)
- Journey (progress, struggles, insights)

---

## Core Loop

notes → exercises → solutions → mistakes → insights → content

---

## Posting Rhythm

- 1 LinkedIn post per week
- 1 Medium article per week
